Attributes <modal-clab>

Name Type Values Default
title string any valid string 'Modal title'
visible boolean true/false false
primary string any valid string none
secondary string any valid string none

Events <modal-clab>

Name Emitted when Custom data
modal-primary Click on the primary button N/A
modal-secondary Click on the secondary button N/A
