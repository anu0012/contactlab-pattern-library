Input calendar

Syntax

<calendar-clab></calendar-clab>

Attributes <calendar-clab>

Name Type Values Default
label string any valid string none
value string a date format string none
disable boolean true/false false
inline boolean true/false false
options object a Rome options object none
placeholder string any valid string none
type string none, 'success', 'warning', 'error', 'info', 'disabled' none

Events <calendar-clab>

Name Emitted when Custom data
datechange Date is changed from the widget or in the linked text input
  • date: the value of the linked text input
  • dateISO: the value of the linked text input formatted as ISO

Methods <calendar-clab>

Name Arguments Returns
getRomeInstance() N/A the Rome instance for the current component

Select

Syntax

<elekti-mer label="Select label" options='["Option1", "Option 2", ..."]'></elekti-mer>

Attributes <elekti-mer>

Name Type Values Default
label string any valid string none
name string any valid string 'elekti'
options array [{value: 'A', label: 'Option 1'}, ...] [{value: 'A', label: 'Option 1'}, {value: 'B', label: 'Option 2'}]
default number any valid number none
value object {value: ' ', label: ' '} none
open boolean true/false false
no-search boolean true/false false
no-results string any valid string 'No results found'
optionsFn function a function returning a Javascript promise that returns the options array non

Events <elekti-mer>

Name Emitted when Custom data
change When a new value is selected from dropdown
  • newValue: the object representing the new value that has been selected
  • oldValue: the object representing the previous value selected
