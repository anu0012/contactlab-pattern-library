UI components

Buttons

Buttons are used as triggers for actions. They are used in forms, toolbars, dialog footers and as stand-alone action triggers. Buttons should rarely, if ever, be used for navigation.
Actions are operations that are performed on objects.
Navigation changes the screen or view or takes you to another context in the application.

Basic buttons

Button Button Button Button Button 
<!-- Standard button -->
<button class="btn">content</button>
<!-- disabled button -->
<button class="btn" disabled>content</button>
<!-- Standard button with icon -->
<button class="btn"><i class="clab clab-icon-sign"></i>content</button>
<!-- Full button -->
<button class="btn full">content</button>
<!-- Flat button -->
<button class="btn flat">content</button>
<!-- Standard button -->
<button-clab>content</button-clab>
<!-- disabled button -->
<button-clab disabled>content</button-clab>
<!-- Standard button with icon -->
<button-clab icon="clab clab-icon-sign">content</button-clab>
<!-- Full button -->
<button-clab appearance="full">content</button-clab>
<!-- Flat button -->
<button-clab appearance="flat">content</button-clab>
Name Type Values Default
appearance string none: bordered button
'full': button filled with background color
'flat': button without borders 		none
icon string any valid icon class none
size string none: regular size button
'small': smaller button		 none
disabled boolean true/false false
