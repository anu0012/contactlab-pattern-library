Alerts

Syntax

<alert-clab title="lorem ipsum" visible> content </alert-clab>

Attributes <alert-clab>

Name Type Values Default
title string any valid string 'Title'
type string 'success', 'warning', 'error', 'primary', 'secondary' 'success'
visible boolean true/false false
labels object primary: any valid string
secondary: any valid string 		primary: 'Confirm'
secondary: 'Cancel'

Events <alert-clab>

Name Emitted when Custom data
primary Click on the primary (confirm) button N/A
secondary Click on the secondary (cancel) button N/A
