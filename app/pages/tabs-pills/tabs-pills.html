Tabs & pills

Syntax

<tabs-clab active="1">
	<div class="tab-content"> tab 1 content </div>
	<div class="tab-content"> tab 2 content </div>
</tabs-clab>

Attributes <tabs-clab>

Name Type Values Default
labels array array of strings 'Tab [index]'
pills boolean true/false false
active number index of the tab to be displayed at first 0
