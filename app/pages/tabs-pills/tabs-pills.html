UI components

Tabs & pills

Tabs & pills are used for alternating between views within the same context, not to navigate to different areas. For navigation refer to the horizontal navigation guidelines.

Tabs

Tab 1 content
Tab 2 content
Tab 3 content
 
<tabs-clab labels='["Tab 1", "Tab 2", "Tab 3"]'>
	<div class="tab-content">Tab 1 content</div>
	<div class="tab-content">Tab 2 content</div>
	<div class="tab-content">Tab 3 content</div>
</tabs-clab>
Name Type Values Default
labels array array of strings 'Tab [index]'
active number index of the tab to be displayed from the start 0
pills boolean true/false false
