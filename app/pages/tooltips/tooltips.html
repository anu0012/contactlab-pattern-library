Tooltips

Tooltips can describe a UI element in a little bit more detail when users deliberately hover on an element.
Default tooltip Primary tooltip Secondary tooltip Info tooltip Success tooltip Warning tooltip Error tooltip 
<tooltip-clab content="Tooltip content" tt-type="primary"></tooltip-clab>
Name Type Values Default
content string any valid string 'The content'
type string any custom class for the span element none
tt-type string none, 'primary', 'secondary', 'info', 'success', 'warning', 'error' none
