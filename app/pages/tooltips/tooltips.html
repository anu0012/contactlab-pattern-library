Tooltips

Syntax


		<tooltip-clab content="Here goes the tooltip content"> span content </tooltip-clab>

Attributes tooltip-clab

Name Type Values Default
content string any valid string 'The content'
type string any custom class for the span element none
tt-type string none, 'primary', 'secondary', 'info', 'success', 'warning', 'error' none
