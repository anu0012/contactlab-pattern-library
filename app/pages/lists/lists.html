Lists

Syntax


		<ol class="default">
			<li></li>
		</ol>
	


		<ul class="default">
			<li></li>
		</ul>

Attributes <ol>

Name Type Values Default
class string none: without numbers
'default': with numbers 		none

Attributes <ul>

Name Type Values Default
class string none: without bullet points
'default': with bullet points 		none

Examples

Ordered list

  1. Lorem ipsum dolor.
  2. Deserunt, nam enim.
  3. Vel, expedita, fuga.
  4. Possimus iusto, repellat!
  5. Possimus, ratione, officia!
