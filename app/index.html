Pattern Library
Open source UI components and visual style guide to create consistency and beautiful user experiences.
In order to achieve this, we need to have a single point of reference for all the patterns we use, so that everyone internally can refer to them.
We also need a shared vocabulary of UI elements and code, to allow teams to communicate with each other in the most efficient way.

We created this pattern library with several goals in mind:

  • to ensure a consistent use of patterns and UI elements
  • to provide guidance on correct usage of the patterns
  • to encourage re-use of code, reducing redundancy to a minimum

It is a “living document” and should be updated when the code underlying the feature changes or when a new pattern is introduced.
